Bonjour,

Voici la procédure d'installation du logiciel de gestion d'établissement scolaire

- Premièrement vous devez retrouver à la racine du projet le fichier epreuve.sql qui constitue le fichier de base de données. 
  Ouvrez votre server de base de données apache puis phpMyAdmin, creez une nouvelle base de données avec le nom epreuve et importer le fichier epreuve.sql.
- Ensuite vous trouverez un fichier .jar qui est l'application. Ouvrez alors votre invite de commande puis tapez la commande suivante:
  java -jar "chemin où se trouve votre .jar". L'application s'éxecute

De plus vous trouverez à la racine du projet deux diagramme, l'un diagramme de cas d'utilisation et l'autre diagramme de classe
